# Spatialytics Reel Generator

Generate promotional reels for GIS projects, data dashboards, and business websites.

This project uses **Next.js + Remotion** to programmatically generate vertical social media videos.

## Features

- Reel generator for social media
- Templates for GIS, websites, and dashboards
- Animated backgrounds
- Logo + CTA overlays
- 1080x1920 vertical video export

## Example Use Cases

• Promote GIS consulting services  
• Create map showcase videos  
• Generate business website promos  
• Produce social media reels for Spatialytics projects  

## Tech Stack

- Next.js
- React
- Remotion
- TailwindCSS
